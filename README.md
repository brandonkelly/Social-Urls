# Social Urls plugin for Craft CMS 3 

Add Social Media Pages to your website

## Installation

#### Requirements

This plugin requires Craft CMS 3.0.0, or later.

#### Plugin Store

Log into your control panel and click on 'Plugin Store'. Search for 'Social Urls'.
